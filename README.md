# ComputeVolume
Computes the volume from vtkPolyData using vtkMassProperties
Supported File Extensions: .ply, .vtp, .obj, .stl, .vtk, .g

Usage:
./ComputeVolume < File Name >


Ex. ./ComputeVolume ../STLFiles/Bridgestone\ Arena.stl
Volume: 10402
    VolumeX: 10157
    VolumeY: 10172.7
    VolumeZ: 10850.3
